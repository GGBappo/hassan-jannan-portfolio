# Hassan Jannan Portfolio

This folder contains your standalone portfolio website.

## Main files

- `index.html`: main page
- `styles.css`: site styling
- `content.js`: your editable content, section headers, links, and project details
- `scripts/main.js`: rendering logic for the content file

## What to edit

1. Open `content.js`.
2. Update text, links, section headers, and project entries there.
3. Replace image paths in `content.js` if you want to swap in real screenshots or a real profile image.

## Project contribution bullets

Each project can include a separate contribution section under the main game description:

```js
detailsHeading: "What I worked on",
details: [
  "Built gameplay systems",
  "Created dialogue logic",
  "Helped lead the project"
]
```

## Preview locally

Open `index.html` directly in your browser. This site is plain HTML, CSS, and JavaScript, so it does not need a dev server.

## GitHub Pages

This site is ready to publish as a GitHub Pages project site.

- Expected project URL format: `https://<github-username>.github.io/<repository-name>/`
- If the repository is named `<github-username>.github.io`, it can publish at the root user-site URL instead.
